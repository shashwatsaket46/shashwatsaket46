## 🚀 Featured Project

### 🍽️ ReservaDirect — Autonomous AI Reservation Agent

Built at **tech@NYU Hackathon (HOF Office)**

ReservaDirect converts Google Calendar events into real-world restaurant reservations by:

- Extracting booking intent from event descriptions
- Discovering and ranking nearby restaurants
- Placing real-time AI voice calls
- Confirming reservations automatically
- Updating calendar events dynamically

📅 Create an Event → 🤖 AI Makes the Call → ✅ Reservation Confirmed
